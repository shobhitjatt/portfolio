# Shobhit Chaudhary Portfolio

Award-worthy developer portfolio built with Next.js 15, TypeScript, Tailwind CSS, Framer Motion, and GSAP.

## Setup

1. Install Node.js 20+ and npm/yarn.
2. Run `npm install`.
3. Run `npm run dev`.

## Features

- Premium dark theme
- Glassmorphism and aurora background
- Animated hero with 3D card effect
- Scroll-triggered section reveals
- Smooth page transitions
- Modern responsive design
